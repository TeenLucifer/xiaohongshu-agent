# 008 Image Lightbox Preview

## 背景

当前工作台中已经有多个图片展示入口：
- 候选帖子详情弹窗中的帖子图片
- 右侧"创作" tab 中的图片生成结果缩略图
- 图片编辑区中的素材图和编辑区缩略图

这些图片目前只能在小尺寸容器里查看，缺少统一的大图预览能力。

## 目标

- 为工作台中的图片展示入口提供统一的大图预览弹层
- 点击图片时打开预览，而不是跳转新页面
- 对多图来源支持左右切换
- 保持各区域已有交互不回归

## 非目标

- 图片缩放、拖拽平移
- 图片下载
- 跨组图片联播
- 后端接口改造

## 用户故事

- 作为运营人员，我希望点击帖子图片后能直接放大查看细节。
- 作为运营人员，我希望在图片结果区里快速切换查看同一组图片。
- 作为运营人员，我希望图片编辑区里的缩略图也能放大，但不影响拖拽和删除。

## 交互规则

### 统一预览弹层

- 点击图片后打开居中的大图预览弹层
- 支持关闭按钮、点击遮罩、`Esc` 关闭
- 支持左右方向键切换图片
- 弹层内不做滚轮缩放和平移

### 多图切换

- 候选帖子详情弹窗中的图片按该帖子 `images[]` 切换
- 图片结果区按当前结果列表切换
- 图片编辑器中的素材图和编辑区图默认按各自当前列表切换
- 第一版不做跨帖子或跨 group 串联切换

### 图片编辑区兼容

- 素材图区单击图片打开预览
- 素材图区拖拽能力继续保留
- 编辑区单击图片打开预览
- 编辑区删除继续只通过右上角删除按钮触发

## 验收标准

- [ ] 候选帖子详情弹窗中的图片可点击放大查看
- [ ] 右侧图片结果区中的图片可点击放大查看
- [ ] 图片编辑区中的素材图可点击放大查看
- [ ] 图片编辑区中的编辑图可点击放大查看
- [ ] 预览弹层可通过关闭按钮、遮罩和 `Esc` 关闭
- [ ] 多图来源支持左右切换
- [ ] 图片编辑区拖拽与删除交互不回归
