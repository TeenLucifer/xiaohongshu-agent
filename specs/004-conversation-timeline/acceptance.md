# 004 Conversation Timeline Acceptance

## 手工验收

1. 进入三栏工作台的中间主栏
2. 确认默认看到的是 user / agent 的简化对话流，而不是日志时间线
3. 查看任意一条 user 消息和一条 agent 回复消息
4. 确认主界面看不到调用过程、工具过程、输入摘要、输出摘要、详细日志和结果摘要块
5. 确认每条消息都采用较小的图标 + 文本结构
6. 确认总结结果和文案结果都能以 agent 回复消息形式出现
7. 确认对话消息字号层级清晰，适合快速扫读
8. 确认顶部说明和输入区不会压过中栏对话主体

## 自动化验收

- 简化对话流渲染测试通过
- 主界面不展示过程信息测试通过
- 多类 agent 回复共存测试通过
- 轻量消息结构测试通过
- 主界面无结果摘要块测试通过

- 当前不接入独立 debug 视图
