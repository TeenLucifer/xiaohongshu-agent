# 007 Image Editor Frontend - Acceptance

## 功能验收

- [ ] 候选图片区正确显示上传图片和帖子图
- [ ] 候选图片区顺序为上传图片优先、帖子图其次
- [ ] 拖拽图片到编辑区成功
- [ ] 编辑区图片显示编号（1, 2, 3...）
- [ ] 点击编辑区图片可移除
- [ ] 移除后编号自动重排
- [ ] 刷新页面后编辑区状态保留

## 交互验收

- [ ] 拖拽时显示视觉反馈
- [ ] 拖入编辑区时目标区域高亮
- [ ] 同一张素材图重复拖入时不会新增第二份
- [ ] 同一张生成图重复拖入时不会新增第二份

## API 验收

- [ ] GET /api/topics/{topic_id}/editor-images 返回编辑区图片
- [ ] PUT /api/topics/{topic_id}/editor-images 更新编辑区图片
