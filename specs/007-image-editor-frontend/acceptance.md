# 007 Image Editor Frontend - Acceptance

## 功能验收

- [ ] 素材区正确显示所有候选帖子的图片
- [ ] 图片按帖子分组，帖子内按图片顺序排列
- [ ] 拖拽图片到编辑区成功
- [ ] 编辑区图片显示编号（1, 2, 3...）
- [ ] 点击编辑区图片可移除
- [ ] 移除后编号自动重排
- [ ] 刷新页面后编辑区状态保留

## 交互验收

- [ ] 拖拽时显示视觉反馈
- [ ] 拖入编辑区时目标区域高亮
- [ ] 同一张素材图重复拖入时不会新增第二份
- [ ] 同一张生成图重复拖入时不会新增第二份

## API 验收

- [ ] GET /api/topics/{topic_id}/editor-images 返回编辑区图片
- [ ] PUT /api/topics/{topic_id}/editor-images 更新编辑区图片
