# 026 Selection Polish Skill Acceptance

## 手工验收

1. 打开某个 topic 的右侧“创作 -> 文案”区
2. 在文案编辑区中选中一段非空文本
3. 确认选区附近出现“AI 润色”入口
4. 点击入口，确认出现提示词输入弹层
5. 输入“更口语一点，保留原意”之类的润色要求
6. 提交后确认进入执行中状态
7. 成功后确认只替换当前选区，不修改其它正文
8. 确认替换后文案仍会自动保存
9. 刷新页面，确认替换结果仍可从 `copy_draft.json` 恢复
10. 查看中间对话流，确认新增一条润色结果消息
11. 清空选区或输入空指令，确认接口不会执行并给出清晰提示

## 自动化验收

- 选区润色专用接口返回结构化 `replacement_text` 测试通过
- 内部 prompt 不进入对话流测试通过
- 空选区 / 空指令 / 空文案校验测试通过
- 文案区滚动与渲染回归测试通过

## 已知限制

- 当前不做 token 级流式选区润色
- 当前 skill 只返回替换文本，不支持结构化 patch
